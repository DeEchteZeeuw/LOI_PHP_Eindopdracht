Mijn naam is Ruben Dalebout. Ik doe deze opdracht aan de LOI tijdens de cursus voor Front-end Developer.
Ik heb op het MBO de opleiding Software Developer afgerond en heb als huidig werk de baan beschrijving Junior Software Engineer.

Voor dit project heb ik gekozen om te coderen in Visual Studio Code. Door middel van verschillende extensies zijn de mogelijkheden naar
mijn mening eindeloos. Ook kan ik hierdoor makkelijk switchen met mijn lokale omgeving en de werkomgeving waarmee ik verbind via SSH.

Het installeren van Visual Studio was zeer gemakkelijk gezien ik het al een geruime tijd gebruik wist ik precies waar ik alles moest downloaden.
Hiermee doel ik zowel op het programma zelf als handige extensies, zoals; minify.

Voor het live draaien van mijn website heb ik gekozen voor Xampp. Toen ik 12 jaar was (zo'n 9 jaar geleden) begon ik al door middel van Youtube
videos Xampp te gebruiken en sindsdien nooit meer verandert van gebruik. Het is zeer makkelijk te draaien en de documentaties zijn altijd heel
duidelijk mocht er toch iets fout gaan. Ook kan ik via Xampp heel makkelijk de ontwikkel omgeving meenemen, omdat het op mijn hardeschijf staat.
